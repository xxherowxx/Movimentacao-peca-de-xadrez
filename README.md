Tema 4 - Movimentação de Peças de Xadrez

Este projeto demonstra os fundamentos da movimentação das peças no jogo de xadrez, utilizando lógica de programação com Python.

✔ Peças implementadas:
- Rei
- Torre
- Bispo

Execute o arquivo `movimentacao.py` para ver os movimentos possíveis a partir da posição (4,4).
